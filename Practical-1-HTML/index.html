HTML Lists

HTML Lists are used to specify lists of information. All lists may contain one or more list elements. There are different types of HTML lists.

1) Ordered List or Numbered List (ol)

Following is an example of Ordered List

  1. <ol> tag defines the start of the list.
  2. <li> tag defines the list items.

2) Unordered List or Bulleted List (ul)

Following is an example of Unordered List

  • <ul> tag defines the start of the list.
  • <li> tag defines the list items.

3) HTML Description List or Definition List

Following is an example of Description List

<dl> tag
- defines the start of the list.
<dt> tag
- defines a term.
<dd> tag
- defines the term definition.

HTML Table

HTML table tag is used to display data in tabular form (row * column). There can be many columns in a row.

We can create a table to display data in tabular form, using <table> element, with the help of <tr> , <td>, and <th> elements.

  1. Table Row is defined by <tr> tag.
  2. Table Header is defined by <th> tag.
  3. Table Data is defined by <td> tag.

HTML Video Tag

HTML 5 supports <video> tag also. The HTML video tag is used for streaming video files such as a movie clip, song clip on the web page.

Currently, there are three video formats supported for HTML video tag:

  1. mp4
  2. webM
  3. ogg
